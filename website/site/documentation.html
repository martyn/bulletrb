Documentation

Shattered Ruby builds upon the efforts of Ogre3D and Ogre.rb. It is important to have a grasp of these libraries as well as Shattered Ruby's abstractions.

Shattered Ruby resources:

  • RDocs

Ogre and Ogrerb resources:

As the Ogre.rb project points out:
Ogre.rb [and Shattered] will not attempt to document the Ogre public API. This library assumes decent working knowledge of Ogre itself, though if you need help Ogre has a very good set of documentation available at these places:

  • Ogre.rb RDocs
  • Ogre Wiki
  • Ogre Manual
  • Ogre API reference
